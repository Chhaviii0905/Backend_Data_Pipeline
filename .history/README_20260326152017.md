#Ba

A data pipeline with 3 Docker services: Flask API mock server, FastAPI ingestion pipeline, and PostgreSQL database.

## Prerequisites

- Docker Desktop (running)
- Python 3.10+
- Git
- docker-compose

## Project Structure

```
project-root/
├── docker-compose.yml
├── README.md
├── mock-server/
│   ├── app.py
│   ├── data/customers.json
│   ├── Dockerfile
│   └── requirements.txt
└── pipeline-service/
    ├── main.py
    ├── models/customer.py
    ├── services/ingestion.py
    ├── database.py
    ├── Dockerfile
    └── requirements.txt
```

## Setup and Running

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Backend_Data_Pipeline
   ```

2. Start all services:
   ```bash
   docker-compose up -d
   ```

3. Verify services are running:
   ```bash
   docker-compose ps
   ```

## API Testing

### Flask Mock Server (Port 5000)

- Health check: `curl http://localhost:5000/api/health`
- Get paginated customers: `curl "http://localhost:5000/api/customers?page=1&limit=5"`
- Get single customer: `curl http://localhost:5000/api/customers/CUST001`

### FastAPI Pipeline Service (Port 8000)

- Health check: `curl http://localhost:8000/api/health`
- Ingest data from Flask: `curl -X POST http://localhost:8000/api/ingest`
- Get paginated customers from DB: `curl "http://localhost:8000/api/customers?page=1&limit=5"`
- Get single customer from DB: `curl http://localhost:8000/api/customers/CUST001`

## Services Description

### Flask Mock Server
Serves customer data from JSON file with pagination support.

### FastAPI Pipeline Service
Ingests data from Flask using DLT library and stores in PostgreSQL with upsert logic.

### PostgreSQL
Stores customer data with the following schema:
- customer_id (VARCHAR(50) PRIMARY KEY)
- first_name (VARCHAR(100) NOT NULL)
- last_name (VARCHAR(100) NOT NULL)
- email (VARCHAR(255) NOT NULL)
- phone (VARCHAR(20))
- address (TEXT)
- date_of_birth (DATE)
- account_balance (DECIMAL(15,2))
- created_at (TIMESTAMP)

## Stopping Services

```bash
docker-compose down
```

## Development

```bash
docker-compose up --build -d
```
